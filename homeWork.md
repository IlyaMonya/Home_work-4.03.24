# Подсказки по командной строке

## Команды

Команда отображения текущей дериктории
```sh
pwd
```

Листинг текущей директории 
Windows:
```sh
dir
```
Linux, MacOs:
```sh
ls
```
## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочками(*) 

Например:

 *курсив* или _курсив_
